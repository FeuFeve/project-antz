# project-antz

Recreation of the "Fourmizzz/Antzzz" online web game in a desktop game made with Unity... More features expected to be developed.

Fourmizzz (FR) : http://www.fourmizzz.fr/
Antzzz (EN) : http://www.antzzz.org/
